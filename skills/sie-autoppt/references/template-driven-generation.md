# Template-Driven Generation (SIE)

本文档沉淀了“按固定模板生成”的稳定策略，用于避免比例错乱、目录错位、颜色漂移与结尾页丢失。

## 1. 适用场景

- 用户提供固定企业模板（`.pptx`）
- 要求首尾固定，中间按内容自动生成
- 要求章节目录重复出现，并高亮当前章节

## 2. 页面角色约定（当前模板）

- 第1页：欢迎页（固定，不改）
- 第2页：主题页（保留版式，仅替换标题）
- 第3页：目录页（可重复插入，按章节高亮）
- 第4页：正文母页（空白正文模板，作为正文克隆来源）
- 最后一页：感谢页（固定，不改，且必须在末页）

> 若模板发生更新，必须先重新识别页面角色后再生成。

## 3. 硬规则

1. 任何正文页必须从“正文母页”克隆，不允许从历史示例页克隆。
2. 目录页复用模板目录页，不允许重绘目录骨架。
3. 当前章节标题高亮色固定：`RGB(173, 5, 61)`。
4. 非当前章节标题统一为目录常规色（当前模板为 `RGB(184, 196, 201)`），避免历史高亮残留。
5. 结尾页必须在最后一页，禁止插到中间。
6. 标题和正文统一字体：`Microsoft YaHei`。
7. 输出路径固定桌面，文件名必须带毫秒级时间戳。

## 4. 输出序列（章节模式）

推荐序列：

1. 欢迎页
2. 主题页
3. 目录（章节1高亮）
4. 正文1
5. 目录（章节2高亮）
6. 正文2
7. 目录（章节3高亮）
8. 正文3
9. 感谢页

## 5. 防错要点

- 禁止用固定坐标“重画”目录页元素，目录页只改章节标题文本。
- 目录页由高亮版克隆后，必须执行“先全量重置非当前色，再设置当前红色”的两步上色。
- 文本替换优先“保留 run 样式”，避免段落位移和前缀残留。
- 正文写入遵循“内容不溢出”原则：短句化、卡片化、必要时降字号。
- 每次输出后至少做一轮结构检查：
  - 末页是否为感谢页
  - 目录高亮是否只有当前章节
  - 正文页是否同构（由正文母页克隆）
