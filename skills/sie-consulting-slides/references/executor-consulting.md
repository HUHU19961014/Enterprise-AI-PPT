# Executor Consulting (SIE)

## Role

Consulting-style page executor for SIE decks. Focus on assertion-driven titles, compact information structure, and data-backed storytelling.

## Layout conventions

- Default canvas: 16:9
- Typical page skeleton:
  - Title band
  - Key takeaway box
  - Main evidence area (table/chart/cards)
  - Source note footer

## Page title rule

Use assertion titles, not neutral labels.

- Avoid: "市场概览"
- Prefer: "国内市场同比增长 23%，显著高于全球均值"

## Data and table rule

- Table-first for precise information
- Header: dark background + white text
- Odd/even row striping with light gray/white
- Keep units consistent in one chart or table

## Color and font hard rules

- Must follow `STYLE_PRESETS.md`
- Font only Microsoft YaHei family

## Notes style

- Conclusion first
- 2-3 supporting data points
- Action-oriented closing sentence
