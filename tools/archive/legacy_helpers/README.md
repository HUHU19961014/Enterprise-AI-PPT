# Legacy Helpers Archive

These scripts are kept for traceability, but they are no longer part of the recommended workflow.

## Archived Scripts

- `html_to_ppt_uat.py`
  Legacy standalone HTML-to-PPT rebuild experiment. Superseded by the current `sie_autoppt` pipeline.

- `template_poc_generate.py`
  Old compatibility wrapper kept only as historical context. Use `python main.py ...` or `python .\tools\sie_autoppt_cli.py ...` instead.

- `bootstrap.ps1`
  Environment-specific bootstrap helper for an older external `ppt-master` workspace.
