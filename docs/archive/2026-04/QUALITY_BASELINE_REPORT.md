## 核心问题汇总

### 高频问题（出现 3 次以上）

1. **标题过长**（4/5 样例）
   - 尾页标题普遍在 22-27 字
   - 建议阈值：≤20 字
   - 当前 quality_checks.py 的阈值是 24 字（warning）和 32 字（high warning）

2. **标题目录化**（1/5 样例，但影响大）
   - 02_project_solution 的 4 个标题都是目录词
   - 缺少结论导向和判断性

### 中频问题（出现 1-2 次）

3. **缺少数据支撑页**（2/5 样例）
   - 01_management_report 和 05_monthly_business_review 都提到缺少关键数据
   - 建议增加数据可视化页

4. **个别 bullet 略长**（2/5 样例）
   - 部分 bullet 在 25-27 字
   - 建议阈值：≤35 字（当前 quality_checks.py 是 40 字）

## 质量闸门建议

基于以上分析，建议设置以下质量闸门：

### Error 级别（必须拦截）

1. **标题长度**：中文标题 >28 字（严重溢出风险）
2. **bullet 数量**：title_content 的 bullet <2 或 >7
3. **bullet 长度**：单条 bullet >50 字（严重过密）
4. **结构异常**：首页不是 section_break，或尾页不是 title_only/section_break

### Warning 级别（记录但不拦截）

1. **标题长度**：中文标题 >20 字（建议精简）
2. **bullet 长度**：单条 bullet >35 字（建议压缩）
3. **two_columns 不平衡**：左右 bullet 数量差距 >3
4. **目录化标题**：标题包含"背景"、"问题"、"方案"、"路径"等目录词（需要 NLP 检测）
