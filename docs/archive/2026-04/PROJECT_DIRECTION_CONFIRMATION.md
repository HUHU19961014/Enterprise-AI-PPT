# 项目方向确认说明

## V1 目标

当前版本的核心目标不是“生成更好看的 PPT”，而是“先稳定产出高质量结构”。

系统核心输出从：

- `PPT 文件`

调整为：

- `Structure JSON`
- `DeckSpec JSON`
- 最终 PPT 仅作为结构落地后的导出结果

统一原则：

- 先生成结构，再生成内容
- 先保证逻辑，再考虑美化
- 先验证结构质量，再扩展模板和 UI

## 本阶段暂停事项

以下方向不作为当前迭代优先级：

- 新增 PPT 皮肤主题
- 动画效果
- 多模型切换
- 复杂编辑器
- 花哨首页
- 一次性生成完整长文案

## V1 成功标准

- 结构输出稳定为合法 JSON
- 能给出明确 `core_message`
- 能产出 3-5 个一级论点
- 每个一级论点下有 2-4 个支撑论据
- 内容层服从结构层，不反向篡改大纲
- PPT 导出无严重排版事故

一句话判断标准：

> V1 首先验证“结构质量”，而不是“页面是否足够炫”。
