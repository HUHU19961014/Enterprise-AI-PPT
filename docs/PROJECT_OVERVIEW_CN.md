# Enterprise-AI-PPT 项目一页介绍

## 项目定位

`Enterprise-AI-PPT` 是一个面向企业汇报场景的 AI PPT 生产引擎，用于把零散需求、业务材料和结构化内容快速转成可编辑的 PowerPoint 文件。

## 解决的问题

- 传统 PPT 制作耗时长，结构与表达质量高度依赖个人经验
- AI 直接生成文本或图片，难以进入正式企业交付流程
- 团队缺少统一模板、统一目录逻辑和统一页面风格
- 从主题到可交付 PPT 中间缺少标准化中间层

## 核心方案

1. 输入主题、Brief、结构化内容，必要时兼容导入 HTML
2. AI 先完成需求澄清与页面规划
3. 输出 Deck JSON 等中间结构
4. 由本地确定性引擎完成语义编译和主题化渲染，输出 PPTX
5. 最后做少量人工润色并交付

## 当前能力

- 自然语言主题生成 PPT
- HTML 内容兼容导入
- 结构化 JSON 生成 PPT
- 语义编译与本地确定性渲染
- 页数范围控制与动态推断
- OpenAI 兼容模型、本地网关

## 业务价值

- 降低 PPT 生产时间
- 提高结构化表达一致性
- 保留企业模板与品牌风格
- 输出可编辑文件，便于继续打磨
- 便于沉淀企业自己的汇报生产流程

## 适用场景

- 管理层汇报
- 客户提案
- 解决方案介绍
- 行业研究
- 项目阶段汇报
- 内部培训材料

## 当前输出物

- `.pptx`
- `.deck.json`
- `.outline.json`
- 渲染日志与质量检查结果

## 推荐展示话术

一句话可以这样讲：

“这个项目不是单纯让 AI 帮我们写 PPT，而是把企业汇报从需求理解、结构规划到模板化输出串成了一条可复用的生产链路，最终交付的是可继续编辑的正式 PPT 文件。”
