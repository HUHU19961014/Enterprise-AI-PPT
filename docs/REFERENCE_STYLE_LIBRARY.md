## 使用原则

- 首尾页、封面、目录页继续严格沿用模板。
- 正文优先选用参考库页型，而不是重新画一套通用盒子。
- AI 的发挥空间放在内容提炼、页型匹配、局部精简和文案重写，不放在脱离参考稿的自由排版。
- 当输入内容与现有参考页型不匹配时，先新增参考样式，再考虑通用兜底渲染。
